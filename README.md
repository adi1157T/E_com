Project Brief — E_com
Type: Full Stack E-Commerce Web Application
Stack:

Frontend: HTML, CSS, JavaScript, React, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL + Prisma ORM
Auth: JWT + bcrypt
Images: Cloudinary
Version Control: Git + GitHub
Deployment: Vercel (frontend) + Render (backend)

Core Features to Build:

Register, Login, Logout with JWT auth
Role based access — Customer and Admin
Product listing, search, product detail page
Admin can add, edit, delete products with images
Cart — add, remove, total calculation
Order placement and order history
Admin can view and manage all orders
Clean, responsive, beautiful UI — no compromise on design

Standard: Less features but everything we build works completely and looks perfect.
Timeline: 20 days. 4-5 hours daily. GitHub commit every night.
